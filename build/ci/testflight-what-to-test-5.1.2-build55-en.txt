TestFlight — What to Test (5.1.2 build 55)

Language switching (regression fix)
• Open the app and switch language to Ukrainian, Chinese, or Japanese from the footer (tap the language label).
• UI must stay usable — no yellow warning icons instead of menus or filters.
• Switch to another language or tap Reset to English at the bottom of the language list.
• Force-quit and reopen the app — selected language should load without breaking the UI.

Wallet name with spaces
• Create or restore a wallet named Arqma Wallet or My Wallet (with a space).
• It should succeed (stored as Arqma_Wallet / My_Wallet) — no Invalid wallet name error.
• Invalid characters (e.g. /, ..) should still show a clear validation message before submit.

Sync after backgrounding (iOS)
• Open a password-protected wallet and start blockchain scan/sync.
• Background the app for several minutes (browse other apps), then return.
• With Face ID enabled: sync should resume after biometric unlock when possible.
• Without Face ID (or if biometric is cancelled): a Continue syncing password dialog should appear automatically — enter the wallet password and sync should resume.
• Cancel the dialog — return to the wallet screen; after a few seconds or another background/foreground cycle, the dialog should be offered again (not stuck forever).
• Wrong password shows an error — re-enter the correct password and sync should continue.
• Remote node status in the footer should reconnect without requiring a full app reinstall.

Sync progress readability
• While scanning, footer and transaction list progress text should be easier to read (brighter) on the dark theme, including when checking the app from the app switcher.

General smoke test
• Open wallet → Transactions / Send / Receive.
• Change language once and return to English.
• Create a new test wallet with a spaced name, then open it.

Please report: language menu stuck / warning triangles, cannot switch back from UA/ZH/JA, sync frozen after background, resume dialog never reappears after cancel, remote node stuck on Connecting…, or Invalid wallet name for normal names with spaces.

Build note: build 55 — resume password dialog after background, store-before-reopen for wallet safety, full locale strings for all languages, cancel/retry resume UX.
