Getting Started

1. Run a synced Arqma daemon and point config.json field daemon_url at its JSON-RPC.

2. In Arqma-Wallet, enable the built-in solo pool and turn off “Mimic public pool reservation” so templates use reserve_size 1 — this stamps the coinbase extra fingerprint this site scans for.

3. Tune start_height and fingerprint fields if no rows appear (see README).
